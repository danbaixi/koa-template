## Koa 快速开发模板

此模板可以快速搭建一个 Koa 项目，包括项目目录结构、热更新、路由、日志、异常捕获、响应封装等功能。

## 使用

1、克隆此仓库

```shell
git clone https://github.com/danbaixi/koa-template.git
```

2、安装依赖

```shell
cd koa-template
npm install
// or
pnpm install
```

3、运行

```shell
npm start
// or
pnpm start
```

默认端口 3000，访问`localhost:3000`返回 JSON 即可。
